Tôi đang học về Git và GitHub

Những bước để push code lên GitHub

Bước 1:

Tạo ra dự án mới trên github đây là phần remote repo

Bước 2:

Tạo dự án trên máy tính dùng để code và lưu trữ cục bộ

Bước 3:

Khởi tạo git: để khởi tạo một kho lưu trữ git cục bộ - Local repo

  • Bước 3.1:

    Mở terminal

  • Bước 3.2:

    config mặc định chọn git bash khi mở terminal

  • Bước 3.3:

    git status : để kiểm tra trạng thái git

  • Bước 3.4:

    git init : khởi tạo một kho lưu trữ Local

Bước 4:

add thư mục dự án vào staging area chờ commit

git add . : add tất cả thư mục
git add tên_thư_mục : add lẻ thư mục, tệp

Bước 5:

Commit và đẩy code lên GitHub

  • Bước 5.1

    git commit -m "first commit" : sau khi commit code đẩy vào local repo chờ push lên GitHub
    - first commit : là tên trong lịch sử

  • Bước 5.2

    git branch -M main : do trên GitHub là main mà trong máy tính cục bộ là master nên cần cập nhật lại

  • Bước 5.3

    git remote add origin Đường_dẫn_đến_dự_án : Remote đến máy chủ git

  • Bước 5.4

    git push -u origin main : push code lên máy chủ Git
